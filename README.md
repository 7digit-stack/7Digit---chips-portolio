## 🚀 Overview

Haldiram's Chips Experience is a modern brand-focused website that showcases different chip flavors through interactive 3D product presentations, smooth animations, and engaging storytelling.

The project is designed to provide a premium digital experience that reflects the quality, taste, and excitement of the Haldiram's brand.
